---
title: Quasar
description: Zero-copy, zero-allocation Solana programs with Anchor-level developer experience.
---

Quasar is a Solana program framework that combines zero-copy account access, zero-allocation dispatch, and familiar Anchor-style macros. Programs built with Quasar achieve compute unit costs comparable to hand-written Pinocchio code while keeping the ergonomics of `#[program]`, `#[account]`, and `#[derive(Accounts)]`.
